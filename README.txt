# Budget Allocation Project

**Version 1.0.0**

This program simulates budget allocation. Here, the user inputs the total quantity available to spend. After which, the user plugs in the ratio of the amount on each expense item. If the proportion sum exceeds 100%, the expense input by the user ceases. Finally, the program displays  the final amount spent on each expense item.
